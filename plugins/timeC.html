timeC ​

timeC下面所有的API对应的时间/日期参数都支持 2024-08-08 或 2024/08/8 或 2024-08-08T07:13:07Z 或 Thu Aug 08 2024 08:00:00 GMT+0800格式

javascript
import { timeC } from 'turing-plugin'

getYear ​

获取年份

javascript
const date = '2024-08-08'
const year = timeC.getYear(date)
// 输出2024

getMonth ​

获取月份

javascript
const date = '2024-08-08'
const month = timeC.getMonth(date)
// 输出8

getDate ​

获取日期

javascript
const date = '2024-08-08'
const date = timeC.getDate(date)
// 输出8

getDay ​

获取周 （1~6代表周一到周六 0代表周日）

javascript
const day1 = timeC.getDay('2024-08-08')
// 输出4
const day2 = timeC.getDay('2024-08-11')
// 输出0

getHours ​

获取小时

javascript
const time = '2024-08-08 14:52:55'
const h = timeC.getHours(time)
// 输出14

getMinutes ​

获取分

javascript
const time = '2024-08-08 14:52:55'
const m = timeC.getMinutes(time)
// 输出52

getSeconds ​

获取秒

javascript
const time = '2024-08-08 14:52:55'
const s = timeC.getSeconds(time)
// 输出55

format ​

日期/时间转化

format(time, type)

参数：

  1. time 需要转换的时间或日期
  2. type 转换生成的格式，具体的格式取值见下面的表格
javascript
const time = '2024-08-08T07:13:07Z'
console.log(timeC.format(time, 'YYYY-MM-DD')) // 输出2024-08-08
console.log(timeC.format(time, 'YYYY-MM-DD hh:mm')) // 输出2024-08-08 15:13
console.log(timeC.format(time, 'YYYY-MM-DD hh:mm:ss')) // 输出2024-08-08 15:13:07
console.log(timeC.format(time, 'YYYY/MM/DD hh:mm:ss')) // 输出2024/08/08 15:13:07
type传参结果
什么都不传2024-08-08 15:13:07
YYYY2024
YYYY-MM2024-08
YYYY-MM-DD2024-08-08
YYYY-MM-DD hh:mm2024-08-08 15:13
YYYY-MM-DD hh:mm:ss2024-08-08 15:13:07
YYYY/MM2024/08
YYYY/MM/DD2024/08/08
YYYY/MM/DD hh:mm2024/08/08 15:13
YYYY/MM/DD hh:mm:ss2024/08/08 15:13:07

  2. month 月份
javascript
const num1 = timeC.get(2023, 2) // 输出28

isLeapYear ​

判断是否为闰年

isLeapYear(year)

参数：

year 年份

const bool1 = timeC.isLeapYear(2024) // 结果为true
const bool2 = timeC.isLeapYear(2025) // 结果为false

compareDates ​

比较两个日期或时间大小

compareDates(time1, time2)

time1 > time2 返回1

time1 === time2 返回0

time1 < time2 返回-1

javascript
const result = timeC.compareDates('2024-08-08', '2024-08-07') // 结果为1 大于
const result = timeC.compareDates('2024-08-08', '2024-08-08') // 结果为0 等于
const result = timeC.compareDates('2024-08-08 15:13:07', '2024-08-07 15:13:50') // 结果为-1 小于

createRangeByDay ​

生成n天内的日期/时间范围

createRangeByDay(n, withTime)

参数：

  1. n 天数
  2. withTime 是否包含时间 ：true是包含，false或者不传则不包含，仅生成日期范围
javascript
const dateRange = timeC.createRangeByDay(2)        // 结果为 ['2024-08-07', '2024-08-08']
const timeRange = timeC.createRangeByDay(2, true)  // 结果为 ['2024-08-07 00:00:00', '2024-08-08 16:53:32']
